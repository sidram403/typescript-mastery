# 01 - TypeScript Basics

This section covers the fundamental concepts of TypeScript including variables, primitive types, and type inference.

Understanding these basics is essential before moving to advanced TypeScript concepts such as interfaces, generics, and advanced type manipulation.

## Why TypeScript?

TypeScript is a statically typed superset of JavaScript that adds type safety to JavaScript applications.

Key benefits:

- Detect errors during development
- Improve code readability and maintainability
- Enable better IDE support and auto-completion
- Make large-scale applications easier to manage

## Real-world Usage

In real applications (React, Node.js, APIs), TypeScript helps developers define strict contracts for data such as:

- API responses
- Component props
- Database models
- Function parameters
